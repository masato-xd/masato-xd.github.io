printf的使用

发表于 2017-06-26 | 分类于 linux | 浏览 次

printf 命令的语法：

printf format-string [arguments...]

参数说明：

  • format-string: 为格式控制字符串
  • arguments: 为参数列表。

结果为:

printf -v file 是指将输出结果使用file变量存储
通过$file 来调用打印
%02d 显示两位,不足两位整数用0补齐
