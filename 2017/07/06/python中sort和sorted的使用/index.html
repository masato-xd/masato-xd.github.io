python中sort和sorted的使用

发表于 2017-07-06 | 分类于 python | 浏览 次

我们需要对List进行排序，Python提供了两个方法

  • List的成员函数 sort 进行排序
  • built-in函数 sorted 进行排序

两者的区别

iterable: 可迭代对象
cmp: 用于比较的函数，比较什么由key决定
*key: * 列表元素的某个属性和函数进行作为关键字
*reverse: * 排序规则. reverse = True, 默认为False, True是代表倒序
