Linux下的策略路由设置

发表于 2017-08-15 | 分类于 linux | 浏览 次

策略路由
策略性是指对于IP包的路由是以系统管理员根据需要定下的一些策略为主要依据进行路由的。
例如我们可以有这样的策略：”当服务器有两块网卡的时候,A网卡来的数据从A出去,B网卡来的数据从B出去,做到源进源出”
